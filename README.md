# Cybersecurity Awareness Bot - Part 1

## Overview
This is a console-based chatbot application developed in C# as part of Part 1 of the assignment.

The chatbot simulates a simple interactive assistant that helps users learn basic cybersecurity concepts such as password safety, phishing awareness, safe browsing, and malware protection.

The system demonstrates user interaction, input handling, structured programming, and basic chatbot logic using keyword-based responses.

---

## Features
- Personalised user greeting using name input
- Voice greeting on startup (WAV file)
- ASCII art logo displayed at launch
- Keyword-based response system
- Cybersecurity topics (passwords, phishing, malware, etc.)
- Input validation for empty or invalid inputs
- Colourful and structured console UI
- Typing effect for improved user experience

---

## Project Structure
- `Program.cs` → Entry point of the application
- `ChatBot.cs` → Handles user interaction and chatbot flow
- `ResponseHelper.cs` → Contains all chatbot responses and keyword logic
- `DisplayHelper.cs` → Handles console UI, colours, ASCII art, and formatting
- `AudioHelper.cs` → Plays voice greeting at startup

---

## How to Run
1. Open the project in Visual Studio 2022
2. Ensure the WAV file is inside the `Media` folder
3. Build the solution (Ctrl + Shift + B)
4. Run the program (Start button)
